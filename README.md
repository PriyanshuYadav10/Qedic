# 🏢 qedic

**qedic** is a powerful and user-friendly Flutter app for complete organization management. It allows employees to seamlessly **check in/out**, submit **daily reports**, manage **trips, leaves, expenses**, and update their **profiles** — while business owners can review, manage, and approve all activities through the app. Built with a clean and animated UI, qedic simplifies daily workflows and keeps teams connected.

---

## 📝 Description

**qedic** is designed to help businesses and teams manage internal activities efficiently. From shift tracking to expense and leave approvals, the app ensures every action is logged and approved transparently.

With features like **animated UI**, **real-time search**, and **profile management**, it streamlines business operations in an elegant mobile interface.

---

## 🚀 Features

- 🕒 Daily **Check-In / Check-Out** for Shifts
- 🧾 Submit and Track **Daily Reports**
- 🧳 Manage and Log **Business Trips**
- 📝 Apply for and View **Leaves**
- 💰 Add and Track **Expenses**
- ✅ Business Owner/Admin can **Approve or Reject** requests
- 👤 Full **User Profile Management**
- 🔍 **Search by Username**
- 🎨 Clean, Animated, and Responsive UI
- 🔐 Secure and Role-Based Access
